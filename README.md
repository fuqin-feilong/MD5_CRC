
# 2020年新工科联盟-Xilinx暑期学校（Summer School）项目。



## 项目概要
  + 设计的目的：
               利用FPGA提升MD5加密的运算速度
  + 相关知识：
               MD5加密算法、qspi串口通信、esp32编程开发
  + 应用方向：  可将FPGA与传统CPU、MCU建立通信利用FPGA并行计算加速的特性，提升传统处理器的算力。
